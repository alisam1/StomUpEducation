webinars - контентная часть страницы вебинаров на vue.js
trans - контентная часть страницы трансляций на vue.js

*_dist - Бандлы частей на vue.js

Для того, чтобы интегрировать часть нужно:
    1. Открыть папку части в терминале
    2. Запустить npm run build, что создаст бандлы в папке dist
    3. Скопировать эти файлы с заменой в *_dist в папке vue
    4. Вставить 3 js файла из бандла через скрипт в нужный контейнер
    5. Скопировать стили из css бандла в файл для gulp в папке frontend/source/sass/blocks
        Для Webinars: broadcast_dist.scss
        Для Trans: trans_dist.scss
    6. Убрать из css бандла стиль для .container в начале файла, и перенести его на страницу,
        иначе, стили для .container применятся на весь проект, так как gulp называет также
    
Для того, чтобы внести изменения в часть vue нужно:
    1. Открыть часть vue как отдельный проект
    2. Внести изменения
    3. Повторить шаги с интегрированием, так как файлы бандла будут переименованы
        
!!!В папке dist есть файл index.html, но он при открытии ничего не будет показывать,
    чтобы это исправить нужно исправить пути в html, требуется привести каждый путь к виду:
    "./static/...", иначе html не распознаёт пути
